Profiles API

Enriches a name with gender, age, and nationality data via Genderize, Agify & Nationalize.

Base URL /api
Endpoints
POST
/profiles
Create a new profile by name
201 Created  ·  200 if already exists  ·  400 / 422 / 502
▼
Request body
{ "name": "ella" }
FieldNotes
namerequiredNon-empty string. Case-insensitive.
Response — 201
{
  "status": "success",
  "data": {
    "id": "uuid-v7",
    "name": "ella",
    "gender": "female",
    "gender_probability": 0.99,
    "sample_size": 1234,
    "age": 46,
    "age_group": "adult",
    "country_id": "NG",
    "country_probability": 0.85,
    "created_at": "2026-04-01T12:00:00Z"
  }
}
Test with curl
curl -X POST http://localhost:8000/api/profiles \ -H "Content-Type: application/json" \ -d '{"name": "ella"}'
GET
/profiles
List all profiles with optional filters
200 OK
▼
Query parameters
ParamExample
genderoptionalmale, female
country_idoptionalNG, US, GB
age_groupoptionalchild, teenager, adult, senior

{

}
Test with curl
curl http://localhost:8000/api/profiles curl "http://localhost:8000/api/profiles?gender=female&age_group=adult"
